# AI Recipe Generator

An AI-powered recipe generator built with LangChain and Google Gemini.
Given any region of the world, it generates a classic recipe, 
ingredients with exact amounts, and step-by-step cooking instructions.

## Built With
- LangChain
- Google Gemini (gemini-2.5-flash)
- Python

## How to Run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add your `GOOGLE_API_KEY` to a `.env` file
4. Run: `python3 recipe.py`
